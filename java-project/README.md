# Java Project - Система управления библиотекой

## Описание проекта

Проект представляет собой консольное приложение на Java для управления библиотекой книг. Приложение позволяет добавлять книги, выдавать их читателям, возвращать книги, искать по авторам и вести историю операций. Проект разработан для демонстрации работы статических и динамических анализаторов кода.

## Функциональность

### 1. Book (Книга)
Класс, представляющий книгу в библиотеке:
- Хранение информации о книге (ISBN, название, автор, год издания)
- Отслеживание доступности книги
- Форматированный вывод информации

**Класс:** `com.example.Book`

### 2. Library (Библиотека)
Основной класс для управления библиотекой:
- Добавление книг в библиотеку
- Выдача книг читателям
- Возврат книг в библиотеку
- Поиск книг по автору
- Ведение истории выдачи книг
- Логирование всех операций

**Класс:** `com.example.Library`

### 3. FileManager (Менеджер файлов)
Класс для работы с файловым хранилищем:
- Сохранение данных о книгах в файл
- Загрузка данных о книгах из файла
- Экспорт логов в файл

**Класс:** `com.example.FileManager`

### 4. Main (Главный класс)
Точка входа в приложение с интерактивным меню:
- Просмотр всех книг
- Выдача и возврат книг
- Поиск по автору
- Просмотр логов операций
- Сохранение данных

**Класс:** `com.example.Main`

## Особенности проекта

- Использование коллекций (HashMap, ArrayList, List)
- Работа с файлами (FileReader, FileWriter, BufferedReader, BufferedWriter)
- Обработка исключений (IOException)
- Логирование операций
- Поиск и фильтрация данных
- Управление состоянием объектов

## Требования

- Java 11 или выше
- Maven 3.6 или выше

## Сборка проекта

```bash
mvn clean compile
```

## Запуск проекта

```bash
mvn exec:java -Dexec.mainClass="com.example.Main"
```

Или скомпилировать и запустить напрямую:

```bash
mvn compile
java -cp target/classes com.example.Main
```
