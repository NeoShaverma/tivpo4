# Python Project - Система управления задачами

## Описание проекта

Проект представляет собой консольное приложение на Python для управления задачами (To-Do List). Приложение позволяет создавать задачи с приоритетами и дедлайнами, отслеживать их статус, получать статистику и экспортировать данные. Проект разработан для демонстрации работы статических и динамических анализаторов кода.

## Функциональность

### 1. task.py (Задача)
Модуль для представления задачи:
- Класс `Task` - представляет отдельную задачу
- Класс `Priority` (Enum) - приоритеты задач (LOW, MEDIUM, HIGH, URGENT)
- Класс `TaskStatus` (Enum) - статусы задач (TODO, IN_PROGRESS, DONE, CANCELLED)
- Проверка просроченных задач
- Форматированный вывод с эмодзи

**Класс:** `Task`, `Priority`, `TaskStatus`

### 2. task_manager.py (Менеджер задач)
Модуль для управления задачами:
- Добавление, обновление и удаление задач
- Фильтрация задач по статусу и приоритету
- Поиск просроченных задач
- Ведение истории операций
- Сбор статистики по задачам

**Класс:** `TaskManager`

### 3. storage.py (Хранилище)
Модуль для работы с файловым хранилищем:
- Сохранение задач в JSON формат
- Загрузка задач из JSON файла
- Экспорт задач в текстовый формат

**Класс:** `TaskStorage`

### 4. main.py (Главный модуль)
Точка входа в приложение с интерактивным меню:
- Добавление задач с приоритетами и дедлайнами
- Просмотр задач с фильтрацией
- Обновление статусов задач
- Просмотр статистики и истории
- Сохранение и экспорт данных

- Логирование операций
